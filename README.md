# Restaurant Sales Data Analysis Dashboard

## Project Overview
This project is an interactive Excel-based analytics solution designed to track and visualize pizza restaurant sales performance. It processes detailed transaction records to provide actionable insights into customer preferences and operational efficiency.

## Key Features
* **Dynamic Data Modeling:** Structured raw data into tables for automated calculation and scalability.
* **Advanced Visual Analytics:**
  - **Hourly Order Trends:** Identifying peak hours for better staff management.
  - **Sales Segmentation:** Breakdown by Pizza Type (Classic, Supreme, etc.) and Size.
  - **Payment Analysis:** Comparison of online vs. offline transaction volumes.
* **Full Interactivity:** Integrated Slicers with multi-pivot table connections, enabling seamless filtering across Time (Year/Month/Day) and Categories.
* **Performance Tracking:** Metric-driven analysis of individual restaurant locations and delivery durations.

## Technical Skills Applied
* **Data Cleaning:** Refining date/time strings and order details.
* **Pivot Table Mastery:** Summarizing thousands of rows for multi-chart displays.
* **Excel Visualization:** Area charts, Donut charts, and Bar charts with custom styling.
* **Dashboard Logic:** Using Report Connections to synchronize multiple visual elements.

## Business Insights
* **Peak Ordering Times:** Discover exactly what time of day generates the most revenue.
* **Top Performers:** Identify which restaurants and pizza types are driving the most volume.
* **Customer Behavior:** Understand the preferred payment methods and sizing choices of the customer base.
